# Wumpus World - Text Adventure Game

A simple CLI-based implementation of the classic **Wumpus World** logic game written in Python. Navigate through a dangerous 5x5 grid, avoid deadly pits, hunt the terrifying Wumpus, find the hidden gold, and make it back alive!

---

## Game Objectives
1. **Explore** the map starting from position `(Row 0, Col 0)`.
2. **Find the Gold (`G`)** hidden somewhere in the world.
3. **Return safely** to the starting position `(Row 0, Col 0)` to win the game.

---

## How to Play

Run the script using Python in your terminal:
```bash
python main.py
